## Estratégia de Mock de Dados

Os dados foram estruturados para simular diferentes perfis de clientes:

- Cliente recorrente com alto volume de compras
- Cliente ocasional com compras moderadas
- Cliente inativo sem compras recentes
- Cliente novo com poucas compras

Além disso:

- Foram criadas variações de produtos por categoria e preço
- Os pedidos foram distribuídos ao longo de vários meses para permitir análise temporal
- Foi incluída uma inconsistência proposital entre o valor total do pedido e a soma dos itens para análise de anomalias
